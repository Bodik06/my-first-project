# my-first-project
git init — перетворює звичайну папку на вашому ПК на Git-репозиторій.

git clone [URL] — копіює проєкт із GitHub на ваш комп'ютер.
git status — показує, які файли ви змінили, а які ще не зберегли.
git add . — готує всі змінені файли до збереження (додає в "чергу").
git commit -m "опис змін" — робить "знімок" стану коду з вашим коментарем.
git push — відправляє ваші збережені зміни (коміти) з комп'ютера на GitHub.
git pull — завантажує останні зміни з GitHub на ваш комп'ютер (важливо, якщо працюєте в команді)
